# RememberAndroid
This Android app is an example of using Android Jetpack components to develop an Android app.
This application allows you to manage reminders.

### Requirements:
- **Android 5.1**
 
### Features Developed:
- Insert reminder.
- Edit reminder.
- See reminders created.

### Future Features:
- Delete reminder.

### Android Jetpack components:
- Lifecycles.
- LiveData.
- Room.
- ViewModel.
